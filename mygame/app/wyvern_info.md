# Wyvern

## A UI framework built for DragonRuby GTK

#### Features
- CSS like functionality
- Material Theme Color Gen
- Custom grid sizes
- Sizing Features
- Basic shapes (Squares, Circles, Triangles)
- Drop Shadows
- Hover Actions
- Border support (wip)
